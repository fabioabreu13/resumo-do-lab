# resumo-do-lab
Lab Computação em Nuvem DIO
Foi criado o usuário no portal do Azure e exibida as ferramentas que podemos criar.

Nesse novo Lab, foi explicado como criar uma máquina virtual, as disponílidades e as zonas. No armazenanto, explicado que pode ser salvo em outros locais e que isso impacta nos valores.

Nesse último Lab do primeiro módulo, foi explicado os conceitos de nuvem, IaaS, PaaS e SaaS. Exibido o modelo de responsabilidade de cada. No Portal Azure, foi exibido as configurações para criação de uma VM e um SQL database

Foi criado no portal azure um grupo de recursos e logo após a criação, foi realizada a criação da rede virtual para esse grupo de recursos e associado a ele.
